# How to Add Course Descriptions

## Quick Start

1. Open: `courses_to_add_descriptions.csv`
2. For each course:
   - Visit the provider website (click the link)
   - Find the course page
   - Copy the course description/overview
   - Paste it in the CSV (Description column)
3. When done, run: `python3 import_descriptions.py`
